# ÉTAPE 1 : Configuration Base de Données

1. Créer prisma/schema.prisma avec tous les modèles
Créons le fichier schema.prisma dans le dossier prisma comme suit : 
- `cd backend` : pour ce rendre dans le reperetoire `backend/`.
- `npm install prisma @prisma/client` : pour installer prisma.
Dans mon cas je l'avais déjà installer lors de l'initialisation des dépendances de Node.js.

2. Créer la base de données
